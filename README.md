# Set Up

Create a new topics with the following settings:

| Property         | Value         | Comment                     |
| ---------------- | ------------- | --------------------------- |
| Topic Name       | `omc`         | You can name this anything. |
| Replication      | `2`           |                             |
| Partitions       | `30`          |                             |
| Retentions Bytes | `10737418240` | 10GB                        |
| Retentions ms    | `3600000`     | 1 Hour                      |
